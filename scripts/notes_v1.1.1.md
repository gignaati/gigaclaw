Giga Bot v1.1.1 is a CLI polish and branding correction release. It adds a `--version` flag to the CLI, corrects the CLAUDE.md template header to Gignaati branding, and standardises the API route directory name to `[gigabot]`.

---

## What's Changed in v1.1.1

### ✨ New Features

- Added `--version` / `-v` flag to the CLI — `gigabot --version` now prints the current package version

### 🐛 Bug Fixes

- Updated `CLAUDE.md` template with Gignaati branding header
- Added `--version` to `printUsage` help text
- Standardised API route directory name to `[gigabot]` for consistency with package identity
